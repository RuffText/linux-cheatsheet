#Keyboard Shortcuts for linux terminal

| Shortcut                       | Description                                      |
-------------------------------------------------------------------------------------
| ctrl + shift + +               | Increase font size                               |
| ctrl + -                       | Decrease font size                               |
| ctrl + l                       | Clear terminal                                   |
| ctrl + a                       | Go to start of a command                         |
| ctrl + e                       | Go to end of a command                           |
| ctrl + c                       | Stop a running command                           |
| alt + backspace                | Delete whole word                                |
| ctrl + shift + c               | Copy                                             |
| ctrl + shift + v               | Paste                                            |
| up and down arrow keys         | Navigate previous command                        |
| tab                            | Auto complete                                    |
| ctrl + left or right arrow key | Navigate whole words                             |
